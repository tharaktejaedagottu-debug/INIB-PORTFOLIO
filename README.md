# Personal Portfolio Website

## Project Description
This is a personal portfolio website developed using HTML and CSS as part of the Web Development Internship at Intern InfoByte.

The portfolio showcases personal information, skills, education, projects, and contact details in a simple and responsive design.

## Features
- Personal Introduction
- Profile Photo
- About Me Section
- Skills Section
- Education Details
- Projects Section
- Contact Information
- Responsive Design

## Technologies Used
- HTML5
- CSS3
